Job Manager Pro plugin to manage your organization's hiring process.

== Description ==

Job Manager is a plugin for managing job lists and job applications on your WordPress site.
Job Manager Provide all the features which you need to manage your organization's hiring process. It’s Open Source and FREE to use on your Self Hosted WordPress based website.
Job manger allows user to add job Listings, manage their job listings. The applicant can apply for a job by submitting the given application form. Admin will get notification of applicant form via email.

= Features =

* Add a Job.
* Add status, category, location, type, position
* Add company which offers the job.
* Can set expire date of the job.
* Set Employee’s salary.
* The applicant can apply for the job.
* The admin will see the list of applicants in the backend.
* Email notification for admin.
* Developer friendly code — Custom Post Types, endpoints & template files.

== Installation Steps ==

= Automatic installation =

Go to Wordpress Backend admin panel. Click on add new plugin, search for “ic-job-manager” plugin. Once you have found the plugin, click on “install now”. The plugin will get installed and you will ready to use plugin within few seconds. Also, you can check the plugins release notes, change logs, description/information about the plugin and user ratings.
In the search field type "WP Job Manager" and click Search Plugins. Once you've found the plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by clicking _Install Now_.

= Manual installation =

The simplest way of installation is manual installation the user need to download the plugin file and uploading it to the web server via FTP application. 

* Download the plugin file, unzip the file on your computer.
* Upload this file to your “wp-content/plugin” directory with ftp application.
* Active the plugin from plugins menu within the WordPress admin.

= Getting started =

After plugin installation::

1. Create a page called “job” and place the [list-icjm-job] short code in it. This will fetch your job list.
2. Add a new job as per your specification.
3. The Applicant can apply for a job via an application form.
4. Email notification Will be sent to the admin after the applicant submits the form.


== Frequently Asked Questions ==

= How do I setup Job Manager? =
For installation and setup of job manager, please view the "installation" page. Mostly you only need to add “short codes” on new pages.

= Can I apply for multiple jobs at once? =
Yes, you can apply for multiple jobs. Because each job has their uniqueness in this plugin.

= What do I need to fill-out in order to send a Job Request? =
You only need to fill-up given  an application form and submit. Your form will gets submitted.

= Listed job will deleted after  the expired date? =
No, for now the job will be remain on  the same page after date expiration.
